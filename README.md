# Dashboard Interativo em Python

Projeto de dashboard interativo feito com Python, Pandas, Plotly e Dash.

## Funcionalidades

- Leitura de dados CSV
- Dashboard web interativo
- Gráficos dinâmicos
- Seleção de tipos de gráficos
- Visualização de vendas por produto

## Tecnologias usadas

- Python 3
- Pandas
- Plotly
- Dash

## Como executar

Instale as dependências:

```bash
python -m pip install -r requirements.txt

Execute o projeto:
python app.py

Abra no navegador:
http://127.0.0.1:8050

Estrutura do projeto:
dashboard-interativo-python/
│
├── app.py
├── vendas.csv
├── requirements.txt
└── README.md
